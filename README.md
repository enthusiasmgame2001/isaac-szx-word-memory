# 三只熊背单词

## 这是一个能边玩游戏边随便背点单词的mod：

- 所有的引导内容都写进游戏内啦，打开本mod即可进游戏愉快地玩耍~
- 当前词库包含：<初中词库>、<高中词库>、<雅思词库>、<考研词库>、<四级词库>、<六级词库>、<专八词库>
- 对了，为了避免玩家看着单词过于枯燥，有一张图片会在屏幕内边旋转边移动
- 本mod纯属娱乐，请勿较真！如果单词词库的中文解释或答案显示有误，请联系我，我会尽快修复！
